About

模块化

ZERO 拥有非常灵活的模块系统，并立志于打造模块生态圈。

举例而言，Twenty 是我们开发的一个博客引擎，它同时也是 ZERO 的官方模块之一。

安装 twenty 是异常简单的，只需要在终端执行 zero install twenty 即可，zero会自动帮你安装 twenty 即其依赖。

事件流编程架构

ZERO 拥有独特的事件流编程架构，可根据事件流转顺序编程。

ZERO 还提供了相关的开发工具，可以在浏览器中查看的事件执行流程图。

这种独特的架构，更适合大型应用、多人协作开发，同时也可以兼容传统 MVC 方式。

An Aura component represents a unit of a page. An independent block of reusable code, wrapped up so that it works well on its own or together with other components.
