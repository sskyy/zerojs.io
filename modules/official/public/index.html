ADVANCED WEB FRAMEWORK FOR NODEJS

Designed for change

Behavior Based Module System

Zero modules use a behavioral convention rather than a calling convention. Once a module is installed, provided behavior will be available to the system automatically without calling a single line of its api.

Advanced Event System

Using Zero build-in event Class to organize your business logic makes your system more flexible.With Zero Dev Tool installed, an interactive web interface is provided to trace the data flow and the event fire stack of any request.

Fast growing ecosystem

There are already dozens of zero modules that provide various functionality such as blogging, chatting and other common features of Social-Network website. And three official Zero application have been released for you to play with. Have fun!

Quick Start

Let's try to install blog system named Twenty which is based on Zero.

First, install zero as global module.

> npm install zero -g

Secondly, use Zero to create a new application and install Twenty as a module.

> zero new blog

> cd blog

> zero install twenty

Twenty requires a config file so we copy one and then start the app.

> cp modules/twenty/config.sample.js modules/twenty/config.js

> node app

If you have problem with npm installation, you can download the demo on GitHub. For more demos please visit here.
